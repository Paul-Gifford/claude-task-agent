# Example Outputs

These are real, unedited files produced by the Claude Task Agent from a single
high-level goal each. They're checked in so you can see what the agent produces
without running it yourself. Live runs write to [`../outputs/`](../outputs/),
which is gitignored.

| File | Goal given to the agent |
|------|-------------------------|
| [aws_certifications_comparison.md](aws_certifications_comparison.md) | "Research the top 3 AWS certifications and compare them" |
| [4_week_AI_Cloud_portfolio_plan.md](4_week_AI_Cloud_portfolio_plan.md) | "Build me a 4-week plan with one AI or Cloud portfolio project per week" |
| [top_10_highest_paying_ai_jobs.md](top_10_highest_paying_ai_jobs.md) | "Find the top 10 highest-paying AI jobs and their salary ranges" |

Each was generated by the agent's web_search → write_file tool loop.
